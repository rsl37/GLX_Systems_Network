# End-to-End Tests

This directory contains Playwright end-to-end tests for the GLX Civic Networking App.

## Running Tests

```bash
npm run test:e2e
```

## Test Structure

- `basic.spec.ts` - Basic application functionality tests
- Future test files will be added as the application grows

## Configuration

The tests are configured via `playwright.config.ts` in the root of the GLX_App_files directory.
