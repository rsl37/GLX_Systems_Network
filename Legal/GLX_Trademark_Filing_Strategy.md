### Country-Specific Considerations

#### European Union
- **Madrid Protocol**: Single filing covers 27 countries
- **EUIPO**: Generally favorable to software trademarks
- **Timeline**: 12-18 months
- **Cost**: $1,200-$2,000 per class

#### Canada
- **CIPO**: Similar to USPTO system
- **Timeline**: 12-24 months
- **Cost**: $400-$800 CAD per class
- **Language**: English/French options

#### United Kingdom
- **UKIPO**: Post-Brexit independent system
- **Timeline**: 6-12 months
- **Cost**: ¬£170-¬£200 per class
- **Fast-track**: Available for software marks
